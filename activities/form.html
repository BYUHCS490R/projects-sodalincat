Forms

Gender

1. The "action" tells where the form data goes after you click submit, like sending it to a webpage or server to be handled.

2. The "method" shows how the form sends data, like using “GET” or “POST” when you submit it.

3. “GET” puts data in the web link where people can see it. “POST” hides the data and sends it safely to the server.
