CSS

The difference between Inline CSS is written inside the html tags

Embedded CSS is written inside the "style" tag in html file.

External CSS is written in a separate .css file and linked to the HTML page

External CSS is better because it keeps style separate from content, makes the code cleaner, easier to read, and you can use the same CSS file for many HTML pages.

A selector is used to choose which HTML element the CSS will style.
