1.Viewport meta tag important because it makes web pages look good on all devices. It controls how a page fits the screen size, especially on phones and tablets for better viewing.

2. The CSS rule that appears last in the code takes priority. So, if “flex-direction: row;” is written after “column;”, the layout will follow the row direction.

3. Use CSS media queries like @media (max-width: 600px) to apply specific styles for phones. This helps adjust layouts, text, and images for smaller screens.
