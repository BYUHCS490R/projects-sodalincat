1. JavaScript is a programming language that makes websites do things (like buttons, menus, slides). It got its name because it was created when another language called Java was very popular, so they named it “JavaScript” to attract people.

2. We can run JavaScript in the browser (like Chrome, Firefox) and on the computer/server using Node.js.

3. For a long time, jQuery was the most popular.

4. Today, React is the most popular front-end JavaScript library.
