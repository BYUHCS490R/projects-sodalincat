1. Use a table when you need to show data or information that is best seen in rows and columns, like schedules, comparisons, or statistics.

2. Do not use a table for long text, lists, or layouts. Use tables only when data clearly fits into rows and columns, not for decoration.

3. We need extra work when making tables for screen readers. Add captions, headers, and clear labels so people with disabilities can understand the table easily.
